# 💰 Braquages

Les braquages représentent une activité majeure du RP illégal. Ils doivent être préparés, cohérents et respecter les règles du serveur.

---

# 📋 Règles générales

## ⏳ Temps de repos entre braquages

Afin d'éviter le spam de scènes illégales et de permettre aux différents groupes de profiter des activités du serveur :

* Un délai doit être respecté entre chaque braquage.
* Le délai moyen est compris entre **3 et 5 heures**.
* Il est interdit d'enchaîner plusieurs gros braquages à la suite.
* Le staff se réserve le droit d'annuler une scène jugée abusive.

---

## 🚔 Intervention des forces de l'ordre

Lors d'un braquage :

* Les forces de l'ordre peuvent intervenir.
* Les négociations doivent être privilégiées.
* Les braqueurs doivent laisser une possibilité d'interaction RP.
* Une fois le braquage terminé, les forces de l'ordre peuvent engager une poursuite.

---

## 🎯 Objectif RP

Le but d'un braquage est :

* Créer une scène RP intéressante.
* Favoriser l'interaction.
* Générer des négociations.
* Créer des poursuites cohérentes.

Le but n'est pas :

* D'obtenir uniquement de l'argent.
* De rechercher systématiquement la fusillade.
* De provoquer du PvP gratuit.

---

# 🏦 Braquage Pacific Bank

## ⚠️ Conditions

Le braquage de la Pacific est soumis à autorisation et aux conditions du serveur.

Sans les prérequis nécessaires, le braquage ne pourra pas être lancé.

---

## 📦 Équipement requis

Avant de commencer le braquage, l'équipe doit disposer de :

| Objet               | Quantité |
| ------------------- | -------- |
| Thermite            | x21      |
| Perceuse            | x34      |
| Ordinateur portable | x1       |

### Provenance

* Thermite : zones de territoire.
* Perceuses : LTD.
* Ordinateur portable : LTD.

---

## 🎯 Déroulement

Une fois les différentes étapes terminées :

* Le coffre principal devient accessible.
* Le butin peut être récupéré.
* Les forces de l'ordre sont susceptibles d'intervenir à tout moment.

---

## 🚓 Après le braquage

Une fois le butin récupéré :

* Vous devez semer les forces de l'ordre.
* Vous devez quitter la zone.
* Vous devez laisser la situation se calmer avant toute nouvelle activité.

Le retour immédiat sur les lieux est interdit.

---

# 💎 Braquage Bijouterie

## 📦 Équipement requis

| Objet                | Quantité |
| -------------------- | -------- |
| Thermite             | x3       |
| Découpeuse de vitres | x2       |
| Perceuse             | x4       |

### Provenance

* Thermite : zones de territoire.
* Découpeuse : shop illégal.
* Perceuses : LTD.

---

## 💰 Récompense

La bijouterie peut rapporter jusqu'à :

```text
5 000 000 $
```

en argent sale.

Le montant final dépend :

* De la réussite des différentes étapes.
* Du nombre de vitrines récupérées.
* De la réussite globale de la scène.

---

## 🚓 Après le braquage

Après récupération du butin :

* Une fuite doit être organisée.
* Les forces de l'ordre peuvent poursuivre les braqueurs.
* Les joueurs doivent attendre que la pression redescende avant de reprendre une activité normale.

---

# 🚑 Braquage de Fourgon

## 📢 Fonctionnement

Les braquages de fourgon apparaissent aléatoirement sur le serveur.

Lorsqu'un évènement est lancé :

* Une alerte apparaît.
* Les joueurs peuvent se rendre sur place.
* Les forces de l'ordre sont prévenues après un court délai.

---

## 🚔 Forces de l'ordre

La LSPD reçoit l'information environ :

```text
30 secondes
```

après le début de l'évènement.

Les participants peuvent être interpellés pour :

```text
Braquage de fourgon
```

---

## 🔨 Ouverture du fourgon

Pour accéder au contenu :

* Détruire le fourgon.
* Tirer dessus.
* L'endommager jusqu'à son ouverture.

---

## 💰 Récompense

Une fois ouvert :

* Les sacs de billets deviennent accessibles.
* L'argent obtenu est de l'argent sale.

---

## ⚠️ Important

Le but de l'évènement est :

* Détruire le véhicule.
* Sécuriser le contenu.
* Organiser la fuite.

Le but n'est pas :

* Tuer tous les joueurs présents.
* Transformer l'évènement en zone PvP.
* Rechercher uniquement l'affrontement.
